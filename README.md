# 🧠 dopaminea.com

أهلاً بك في **dopaminea.com**، المنصة المخصصة لفهم وتحليل كيمياء الدوبامين وأثره على السلوك البشري والإنتاجية. هذا المشروع يدمج بين المعرفة العلمية وبين تجربة الويب الحديثة.

## 🔬 عن المشروع
يهدف هذا الموقع إلى تقديم محتوى دقيق ومبسط حول "ناقل السعادة والتحفيز". نحن نستعرض هنا:
* **علوم الدوبامين:** كيف يعمل في الدماغ؟
* **التحكم والإنتاجية:** علاقة الدوبامين بعاداتنا اليومية.
* **الوعي الرقمي:** كيف تؤثر التقنية على نظام المكافأة في دماغنا.

## 🛠 التقنيات المستخدمة
* **الهيكلية:** HTML5, CSS3, JavaScript.
* **التصميم:** واجهات عصرية (Minimalist & Modern UI).
* **الاستضافة:** GitHub Pages.
* **الأمان:** HTTPS.
